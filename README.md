# React Typescript Boilerplate

Boilerplate with configured ESLint, Prettier, Stylelint, Editorconfig, Lint Staged & Husky hooks.

### Steps to reproduce
Install all the packages needed to run your react application.
```shell
npm install
```
then
```shell
npm start
```
